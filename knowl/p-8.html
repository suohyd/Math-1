We denote by \(0_n\) the \(n\times n\) matrix with all its entries zero. We call \(0_n\) the zero matrix. We denote by \(I_n\) the \(n\times n\) matrix with all its diagonal entries \(1\text{,}\) and all other (non-diagonal) entries zero. We call \(I_n\) the identity matrix. We denote the set of all \(n\times n\) matrices over \(\mathbb{R}\) by \(M_n(\mathbb{R})\text{.}\) So \(M_{n\times n}(\mathbb{R})=M_n(\mathbb{R})\text{.}\) For any matrix \(A\in M_{m\times n}(\mathbb{R})\) and \(\alpha\in\mathbb{R}\) we denote by \(\alpha A\in M_{m\times n}(\mathbb{R})\) the matrix obtained by multiplying every entry of the matrix \(A\) by \(\alpha\text{.}\)
