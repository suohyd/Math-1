Consider the following maps. In each of the following cases do the following.
  1. Check if the map defined is linear or not.
  2. If the map is linear then find the matrix associated to it as outlined in Definition 1.7.5.
  3. Draw the images of standard basis vectors (see Definition 1.7.2).
  4. If the map is linear and if its associated matrix, \([T]=A\text{,}\) is invertible then find the linear map associated to \(A^{-1}\text{,}\) \(\ell_{A^{-1}}\) (see Remark 1.7.7). Compute \(T\circ\ell_{A^{-1}}\) and \(\ell_{A^{-1}}\circ T\text{.}\)
